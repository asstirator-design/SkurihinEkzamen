День 1 — Навигация, базовые меши и материалы

На этом этапе ты научишься уверенно двигаться в 3D-пространстве, создавать простые объекты (меши), изменять их форму и положение, а также назначать материалы. В конце урока ты соберешь свою первую простую 3D-сцену.

Навигация в 3D-пространстве

Работа в Blender происходит в окне Viewport (центральная часть экрана).

Основные действия

  • Вращение камеры: зажать среднюю кнопку мыши (колесо) и двигать мышь
  • Приближение / отдаление: прокрутка колесика мыши
  • Перемещение (панорамирование): Shift + средняя кнопка мыши

Фокус на объекте

  • Выдели объект
  • Нажми Numpad . (точка)
  • Камера сфокусируется на нем

Выбор и управление объектами

Выбор объекта

  • Левая кнопка мыши

Перемещение, вращение, масштаб

Используются горячие клавиши:

  • G — перемещение (Grab)
  • R — вращение (Rotate)
  • S — масштаб (Scale)

После нажатия:

  • двигай мышь
  • нажми левую кнопку мыши для подтверждения
  • нажми правую кнопку или Esc для отмены

Ограничение по оси

После нажатия G / R / S можно нажать:

  • X — по оси X
  • Y — по оси Y
  • Z — по оси Z

Пример: G → X → двигать только по X

Создание объектов (мешей)

Меш (Mesh) — это базовый 3D-объект, состоящий из вершин, ребер и полигонов.

Добавление объекта

  1. Нажми Shift + A
  2. Выбери Mesh
  3. Выбери нужный объект

Основные примитивы:

  • Cube (куб)
  • Plane (плоскость)
  • Sphere (сфера)
  • Cylinder (цилиндр)

Удаление объекта

  • Выдели объект
  • Нажми X → подтверди удаление

Работа с несколькими объектами

Выбор нескольких объектов

  • Удерживай Shift и кликай по объектам

Дублирование

  • Shift + D → перемести копию

Базовое редактирование формы

Пока используем только Object Mode (режим объектов). Редактирование вершин будет позже.

Но уже сейчас можно:

  • менять размер
  • вытягивать объекты
  • комбинировать формы

Материалы

Материал определяет, как выглядит объект: цвет, отражение и яркость.

Назначение материала

  1. Выдели объект
  2. Перейди в панель справа (иконка шарика)
  3. Нажми New

Изменение цвета

  • В параметре Base Color выбери любой цвет

Применение материалов

  • Каждому объекту можно задать свой цвет
  • Каждому объекту можно задать свой материал

Практика: создание простой сцены

Теперь нужно собрать небольшую локацию.

Шаг 1 — Основа (земля)

  • Добавь Plane (Shift + A → Mesh → Plane)
  • Увеличь масштаб (S)
  • Это будет поверхность

Шаг 2 — Объекты сцены

Добавь несколько кубов:

  • дома
  • блоки
  • любые простые формы

Используй:

  • Shift + D для копирования
  • S для изменения размера
  • G для перемещения

Шаг 3 — Разнообразие

Сделай сцену не одинаковой:

  • разные размеры
  • разное расположение
  • можно немного вращать объекты

Шаг 4 — Материалы

Задай материалы:

  • плоскости — один цвет (например, земля)
  • объектам — разные цвета
  • каждый объект должен визуально отличаться

Шаг 5 — Композиция

Постарайся:

  • не ставить всё в одну линию
  • оставить пространство
  • сделать сцену читаемой

Что важно запомнить

  • Ты работаешь в 3D-пространстве, а не в плоском интерфейсе
  • Почти все действия делаются через горячие клавиши
  • Простые формы можно комбинировать в сложные сцены
  • Материалы сильно влияют на восприятие

Задание

Создай простую 3D-локацию.

Обязательные условия:

  • есть плоскость (земля)
  • есть минимум 5–10 объектов (кубы или другие примитивы)
  • объекты отличаются по размеру и положению
  • у объектов заданы материалы

Итог

  • есть первая 3D-сцена
  • есть понимание, как двигаться в пространстве
  • есть базовый навык работы с объектами
  • есть опыт назначения материалов

На следующем этапе ты начнешь работать с формой объектов и создашь простого персонажа.
